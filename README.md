# redruffle

RedRuffle let's you run AS2 on the command line using Ruffle. Inspired by [redtamarin](https://github.com/Corsaair/redtamarin).

# Usage
Use `cargo` to compile & run RedRuffle, like this:
```
cargo run --release
```

# Example usage
```
>>> for (var a = 0; a < 5; a++) {
... trace(a)
... }
0
1
2
3
4
>>> trace(a)
5
```
Exit the shell using ctrl+c
